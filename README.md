Fed ACH List
============

API to retrieve data from the Fed ACH List.

Pre-requisites
--------------
- Node.js and NPM

Build and Run
-------------

    # start the site locally at http://localhost:3000
    npm start

    # build the docker image
    make build
    
    # build & push the docker image
    make push
    
    # run the server 
    make run
    
    # run the downloader
    make run-downloader
